## About me

I am an Honors Computer Science student at **The University of Texas at Arlington** with a **4.0 GPA**. I enjoy building software that crosses the boundary between code and the physical world, where latency, safety, privacy, hardware, and human behavior all shape the solution.

My current work spans multiplayer VR training, secure robot teleoperation, local conversational AI, and real-time computer vision. I am also a Presidential Scholar, undergraduate researcher, and software team lead.

## Currently building

- **VR de-escalation training:** developing Unity and C# systems for a federally funded, Meta Quest 3 training environment. I am prototyping multiplayer scenarios and integrating a fully local conversational-AI pipeline with Whisper, an Ollama-hosted Gemma model, and Supertonic voice synthesis.
- **[Secure robot teleoperation](https://github.com/tahmidWasif/dualshock-netbird-teleop):** connecting a Windows DualShock 4 controller to an Ubuntu/ROS 2 robotic-arm stack over NetBird and UDP. The Python bridge includes HMAC-SHA256 authentication, replay protection, a dead-man control, configurable mappings, fail-safe timeouts, and automated tests.
- **RoboMaster computer vision:** building an end-to-end YOLOv8n detection pipeline from annotated competition footage and translating target offsets into C++ turret yaw and pitch commands.
